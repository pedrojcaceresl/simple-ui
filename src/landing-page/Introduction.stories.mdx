# Bienvenidos al Storybook de FacturaE!

Este Storybook muestra todos los componentes y características disponibles en FacturaE. Puedes navegar a través de diferentes historias utilizando la barra lateral a la izquierda.

## Para Empezar

Para comenzar, selecciona un componente o característica de la barra lateral, y podrás ver ejemplos y documentación.

Siéntete libre de explorar y experimentar con diferentes componentes y características.
